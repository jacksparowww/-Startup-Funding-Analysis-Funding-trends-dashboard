# Startup Funding Analysis – Funding Trends Dashboard

## Overview
Startup Funding Analysis is an interactive Streamlit dashboard that provides comprehensive insights into Indian startup funding trends. The application enables users to analyze investment patterns, funding rounds, sector performance, city-wise distribution, and investor activities through dynamic visualizations.

## Objectives
- Analyze overall funding trends
- Identify top-funded startups and investors
- Examine sector-wise and city-wise investment distribution
- Evaluate month-on-month and year-wise funding patterns
- Provide interactive startup and investor-level analysis

## Features
- Overall funding metrics (Total, Maximum, Average, Funded Startups)
- Month-on-Month investment trend analysis
- Year-wise total funding analysis
- Top sectors by number of investments and total funding amount
- Top cities by funding activity
- Funding round analysis
- Funding heatmap visualization
- Startup-specific detailed analysis
- Investor-specific detailed analysis

## Tech Stack
- Python
- Streamlit
- Pandas
- Matplotlib
- Seaborn
